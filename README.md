# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.  
This project allows two players to play the classic Tic Tac Toe game in the browser.

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ✨ Features
- Two-player gameplay
- Winner detection
- Draw detection
- Restart game functionality
- Responsive design
